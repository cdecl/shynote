# 키보드 단축키 및 마우스 기능 가이드 (Keyboard & Mouse Guide)

SHYNOTE는 마우스와 키보드를 활용한 직관적이고 강력한 편집 경험을 제공합니다. 특히 **VS Code** 스타일의 단축키와 **고급 마우스 인터페이스**를 지원하여, 사용자가 오직 글쓰기에만 집중할 수 있도록 돕습니다.

## 1. 키보드 단축키 (Keyboard Shortcuts)

SHYNOTE의 단축키는 대부분 **macOS(`Cmd`)**와 **Windows/Linux(`Ctrl`)**를 모두 지원합니다. 아래 표에서는 `Mod`를 운영체제에 맞는 수정 키로 이해하시면 됩니다.

### 글로벌 및 네비게이션
| 기능 | 단축키 (Mac) | 단축키 (Win/Linux) | 설명 |
| :--- | :--- | :--- | :--- |
| **커맨드 팔레트** | `Cmd + P` | `Ctrl + P` | 모든 명령과 노트를 검색하고 실행합니다. |
| **모드 전환 (Edit)** | `Cmd + 1` | `Ctrl + 1` | 에디터 전용 모드로 전환합니다. |
| **모드 전환 (Preview)** | `Cmd + 2` | `Ctrl + 2` | 미리보기 전용 모드로 전환합니다. |
| **모드 전환 (Split)** | `Cmd + 3` | `Ctrl + 3` | 분할 화면 모드로 전환합니다. |
| **닫기 / 취소** | `Esc` | `Esc` | 팔레트, 모달, 검색창을 닫거나 선택 모드를 종료합니다. |

### 에디터 편집 (VS Code 스타일)
| 기능 | 단축키 (Mac) | 단축키 (Win/Linux) | 설명 |
| :--- | :--- | :--- | :--- |
| **수동 저장** | `Cmd + S` | `Ctrl + S` | 변경 사항을 즉시 저장하고 서버와 동기화합니다. |
| **실행 취소** | `Cmd + Z` | `Ctrl + Z` | 마지막 작업을 취소합니다. |
| **다시 실행** | `Shift + Cmd + Z` | `Shift + Ctrl + Z` | 취소했던 작업을 다시 실행합니다. |
| **줄 이동 (위/아래)** | `Option + ↑ / ↓` | `Alt + ↑ / ↓` | 현재 줄 또는 선택된 블록을 위/아래로 이동합니다. |
| **줄 복제 (위/아래)** | `Shift + Opt + ↑/↓` | `Shift + Alt + ↑/↓` | 현재 줄을 위/아래에 복사합니다. |
| **줄 삭제** | `Shift + Cmd + K` | `Shift + Ctrl + K` | 현재 커서가 위치한 줄을 삭제합니다. |
| **들여쓰기** | `Cmd + ]` | `Ctrl + ]` | 현재 줄의 들여쓰기를 추가합니다. |
| **내어쓰기** | `Cmd + [` | `Ctrl + [` | 현재 줄의 들여쓰기를 제거합니다. |

### 검색 및 다중 커서
| 기능 | 단축키 (Mac) | 단축키 (Win/Linux) | 설명 |
| :--- | :--- | :--- | :--- |
| **찾기 (Find)** | `Cmd + F` | `Ctrl + F` | 에디터 내 검색 및 바꾸기 패널을 엽니다. |
| **다음 일치 항목** | `Cmd + G` | `Ctrl + G` | 다음 검색 결과로 이동합니다. |
| **이전 일치 항목** | `Shift + Cmd + G` | `Shift + Ctrl + G` | 이전 검색 결과로 이동합니다. |
| **항목 추가 선택** | `Cmd + D` | `Ctrl + D` | 현재 선택한 단어와 일치하는 다음 항목을 선택(다중 커서)합니다. |
| **전체 항목 선택** | `Shift + Cmd + L` | `Shift + Ctrl + L` | 현재 선택한 단어와 일치하는 문서 내 모든 항목을 선택합니다. |
| **위/아래 커서 추가** | `Cmd + Opt + ↑/↓` | `Ctrl + Alt + ↑/↓` | 현재 위치의 위/아래 줄에 추가 커서를 생성합니다. |

### 마크다운 서식
| 기능 | 단축키 (Mac) | 단축키 (Win/Linux) | 설명 |
| :--- | :--- | :--- | :--- |
| **굵게 (Bold)** | `Cmd + B` | `Ctrl + B` | 선택 영역을 굵게 표시하거나 `**`를 삽입합니다. |
| **기울임 (Italic)** | `Cmd + I` | `Ctrl + I` | 선택 영역을 기울이거나 `*`를 삽입합니다. |
| **링크 (Link)** | `Cmd + K` | `Ctrl + K` | 마크다운 링크 서식 `[]()`을 삽입합니다. |

## 2. 마우스 인터페이스 (Mouse Interactions)

### 에디터 특화 기능
*   **표 편집기 (Table Editor)**: 마크다운 표 영역을 **더블 클릭**하면 팝업 형태의 전용 편집기가 열립니다. 엑셀처럼 행/열을 추가하거나 정렬을 변경할 수 있습니다.
*   **다중 커서 (Multi-Cursor)**: `Option(Alt) + 클릭`을 사용하여 문서의 여러 곳에 동시에 커서를 두고 편집할 수 있습니다.
*   **박스 선택 (Box Selection)**: `Shift + Option(Alt) + 드래그`를 하면 사각형 영역으로 텍스트를 선택할 수 있어, 여러 줄의 앞부분을 한꺼번에 편집할 때 유용합니다.
*   **백링크 네비게이션**: `[[노트 제목]]` 형태의 위키 링크를 클릭하면 해당 노드로 즉시 이동합니다.
*   **이미지 드래그 앤 드롭**: 외부 이미지 파일을 에디터로 드래그하면 자동으로 서버에 업로드되고 마크다운 링크가 삽입됩니다.

### 파일 및 폴더 관리
*   **드래그 앤 드롭 (이동)**: 노트를 드래그하여 특정 폴더 위로 가져가면 해당 폴더로 이동합니다. 폴더를 드래그하여 다른 폴더에 넣으면 하위 폴더로 지정됩니다.
*   **고무줄 선택 (Marquee Selection)**: 노트 리스트의 빈 공간에서 **클릭 후 드래그**하면 사각형 선택 영역이 나타나며, 여러 노트를 한꺼번에 선택하여 일괄 삭제하거나 이동할 수 있습니다.
*   **다중 개별 선택**: `Cmd(Ctrl) + 클릭`을 사용하여 리스트 내의 특정 노트들을 개별적으로 다중 선택할 수 있습니다.
*   **파일 가져오기**: `.md`, `.txt` 파일을 노트 리스트 영역으로 드래그하면 새 노트로 즉시 생성됩니다.

### 레이아웃 및 기타
*   **분할 화면 조절**: Split 모드에서 에디터와 미리보기 사이의 경계선을 드래그하여 화면 비율을 자유롭게 조절할 수 있습니다.
*   **사이드바 제어**: 왼쪽 상단의 메뉴 아이콘을 클릭하거나, 모바일에서 화면 왼쪽 끝을 오른쪽으로 스와이프하여 사이드바를 열 수 있습니다.
*   **최근 노트**: 사이드바 하단의 'Recent Notes' 섹션을 클릭하여 최근에 작업했던 문서로 빠르게 전환합니다.

## 3. 슬래시 커맨드 (Slash Commands)

에디터에서 새 라인이나 공백 뒤에 `/`를 입력하면 자동 완성 메뉴가 나타납니다.

| 명령어 | 설명 |
| :--- | :--- |
| `/date` | 현재 날짜 (YYYY-MM-DD) 삽입 |
| `/time` | 현재 시각 (HH:MM) 삽입 |
| `/datetime` | 현재 날짜 및 시각 삽입 |
| `/file` | 현재 노트의 제목 삽입 |
| `/code` | 코드 블록 템플릿 삽입 |
| `/table-new` | 새 마크다운 표 생성 (표 편집기 실행) |

## 4. 플랫폼별 차이점

| 항목 | macOS | Windows / Linux / Android |
| :--- | :--- | :--- |
| **기본 수정 키** | `Command (⌘)` | `Control (Ctrl)` |
| **보조 수정 키** | `Option (⌥)` | `Alt` |
| **줄 이동/복제** | `Option` | `Alt` |
| **모바일 제스처** | 지원 (스와이프) | 지원 (스와이프) |
| **드래그 앤 드롭** | 즉시 시작 | 길게 누르기 후 시작 (터치 환경) |

---

> **Tip**: 모든 기능을 외울 필요는 없습니다. `Cmd + P`를 눌러 커맨드 팔레트를 열면, 사용 가능한 명령 옆에 해당 단축키가 친절하게 표시됩니다!
